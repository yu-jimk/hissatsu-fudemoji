# 必殺！筆文字

本プロジェクトは、デカ筆と VIVE Tracker を用いて、一画を加えることで別の漢字を完成させる漢字クイズゲームです。
プレイヤーは筆を動かして一画を描き、正しい漢字を完成させるとスコアが加算されます。
ゲームは二画面構成で、上画面のモニターには問題の漢字が表示され、下画面の床にはプロジェクタで投影された解答欄に筆で直接解答する形式になっています。

## 必要環境
- Unity 2019.4.27f1
- SteamVR Plugin (2.7.X系) 
- VIVE Tracker
- デカ筆 (全長135cm)

## ディレクトリ構成
本プロジェクトは絶対パスによる管理をしているため、実機で動作させる場合は以下のディレクトリ構成を再現して動作させてください
```
C:\
└─ Users
   └─ TGS
      └─ Desktop
         └─ fudemoji
            └─ 3D2Y - コピー
               ├─ Assets
               ├─ ProjectSettings
               ├─ Packages
               └─ ...
```
